# Aureon | Modern Banking for Modern Businesses

Aureon is a premium, high-performance fintech platform designed for fast-growing startups. It provides an all-in-one financial operating system that brings together banking, payments, and expenses on a single, cinematic interface.

## ✨ Key Features

- **Global Accounts:** Hold, send, and receive funds in 40+ currencies with zero hidden fees.
- **Instant Transfers:** Settle payments in seconds via SWIFT GPI, ACH, and local rails.
- **Smart Analytics:** Real-time visibility into your runway, burn rate, and projected cash flow.
- **Premium Experience:** Cinematic landing page with interactive 3D-effect mockups and immersive visual storytelling.

## 🚀 Tech Stack

- **Frontend:** React (SPA), Vite, TypeScript
- **Styling:** Tailwind CSS, Framer Motion (for high-fidelity animations)
- **Icons:** Lucide React
- **State Management:** Redux Toolkit

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Abhay-551/major-project.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd major-project/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
